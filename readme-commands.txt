Semana OmniStack

Iniciando node:
npm init -y

Instalando express para app:
npm install express

npm install nodemon -D

Criando app frontend:
npx create-react-app frontend

Instalando banco:
npm install knex
npm install sqlite3

Iniciando banco:
npx knex init

Criando arquivos para criação de tabelas:
npx knex migrate:make create_incidents

Criando banco:
npx knex migrate:latest

Instalando pacote de icones:
npm install react-icons

Criando rotas no frontend:
npm install react-router-dom

Instalando cliente http:
npm install axios
